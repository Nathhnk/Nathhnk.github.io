Mon expérience professionnelle

Stage de 1ère année de BTS SIO – IFCE Strasbourg

Durée : 1 mois

Période : 2 Juin au 4 Juillet 2025

Mission principale : Conception et développement d’une page web dynamique reliée à une base de données.

Détails des réalisations :

  • Analyse des besoins et conception d’un mini cahier des charges.
  • Création d’une base de données (MySQL) pour la gestion des informations affichées sur le site.
  • Développement d’une interface web en HTML, CSS, JavaScript et PHP.
  • Mise en place de requêtes SQL pour afficher, insérer et modifier les données dynamiquement.
  • Tests et présentation du projet au tuteur de stage.

Compétences acquises : Développement web, conception de base de données, interaction front-end / back-end, autonomie et gestion de projet.
